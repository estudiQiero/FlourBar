FlourBar By Miq 2021

Concepto de barra de progreso para web de cliente de inmobiliaria.

Basado en la idea del progreso de la venta de pisos dentro de una promoción, y para incentivar la compra por parte de los clientes, ante la vista de la BARRA de PROGRESO, donde el porcentaje de pisos vendidos es cada vez mayor.

En realidad, la aplicación es una especie de stand alone, ya que no se integra en la web. Solamente la barra de progreso es la que aparecería gráficamente en la página del cliente. Para hacerla efectiva haría falta un backend desde donde controlarla.

Aun tiene algunos fallos que hay que corregir, como que no cmabia la clase "vendido" cuando reseteamos el contenido, o si escribimos alguna letra en lugar de números en los campos vacíos, aparece un NaN q no queda muy bonito.
